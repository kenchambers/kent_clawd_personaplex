# Notion Organization Skill

## Purpose

Help Kent organize his scattered data in Notion into maintainable systems.

## Connection

Notion is connected via MCP server (OAuth authenticated).
- Server: https://mcp.notion.com/mcp
- Auth: OAuth token (stored by Moltbot)

## Capabilities

### Search & Read
- Search across all pages and databases
- Read page content and properties
- Query database entries with filters

### Create & Update
- Create new pages in any workspace
- Add content blocks (text, lists, code, etc.)
- Update database entries

### Organization Support
- Help structure Kent's notes into systems
- Suggest organizational improvements
- Move/link pages for better structure

## Usage Examples

**Find Information:**
- "Find my notes about the GoldGoblin EA"
- "What's in my trading journal from last week?"

**Create Content:**
- "Create a page for today's trading notes"
- "Add an entry to my ideas database"

**Organize:**
- "Help me restructure my trading notes"
- "Create a template for daily trade reviews"

## Philosophy (from SOUL.md)

- Propose simple, maintainable systems
- Don't create complexity Kent won't maintain
- Break overwhelming tasks into steps
